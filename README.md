# tomato-clock-silently
静默版番茄钟PC端桌面实用工具
## 理想功能需求
首先，我希望电脑一旦开机，我可以随时快捷的打开番茄钟计时功能，所以它可能需要一个开机自启动权限，它需要在电脑桌面有一个类似悬浮窗功能，我希望未唤起时它的悬浮窗图案是普罗旺斯番茄正视图，当开始25分钟专注时长计时，图案变成普罗旺斯番茄的横截面（连着果肉和梗竖直从上往下切开的截面，果肉饱满），右侧显示本次循环已专注时长（一次循环为25分钟专注时长+5分钟休息时间）和一个暂停按钮，本次循环已专注时长下方有一个类似奖状的图案，上面显示累计已专注时长，处于25分钟专注时间时，累计已专注时长是实时增加的（理想情况下，我希望能够记录当天0：00-24：00累计专注时长，也就是用户当天每次循环中的专注时间累加，并且我希望有能够记录每天循环次数、专注时长以及每周、每月、每年数据的文件。当25分钟结束开启5分钟休息时间计时，弹出一束鲜花的动态放大鼓励效果，之后悬浮窗图案变为绿色简笔画番茄小人喝水动态图案，由于奖状中记录的是累计专注时长，所以每次5分钟休息时间，奖状内累计专注时长应该是个不变的数字。每当一个30分钟的循环结束，弹出一个绿色森林的动态效果，给出“开启下一个循环计时”和“结束”两个按钮选择。在每次循环中，随时可以选择点击计时框右侧的暂停，当点击暂停后，出现“继续”和“结束”两个功能选择按钮，一旦选择继续，恢复当前计时（专注时间或者休息时间），一旦选择结束，悬浮窗恢复为完整的普罗旺斯番茄图案，其余计时、暂停、奖状等内容消失。本番茄钟全程静音，不涉及任何音效，仅涉及图案。
